# pyramid of doom

A poor and outdated way to nest asynchronous events.

```javascript
doSomething(function(result) {
  doSomethingElse(result, function(newResult) {
    doThirdThing(newResult, function(finalResult) {
      console.log('Got the final result: ' + finalResult);
    }, failureCallback);
  }, failureCallback);
}, failureCallback);
```

The preferable way is to chain them using [[Promise]]s.
